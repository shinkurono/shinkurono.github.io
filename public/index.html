Anomaly detection on a Character Font Images Dataset

I worked on this character font images dataset as part of course requirements.

Preprocessing was done to balanced the classes using SMOTE, which uses a KNN algorithm to create synthetic data.

The data was then transformed into a 2D Numpy array in order for it to be accepted into a CNN. Normalisation was then done to the values in the array.

Afterwards, the data is then passed through a CNN which is then trained and the results evaluated. Hyperparameters were also tuned during this time to ensure the most accurate predictions.

The model attained an AUC score of 0.991 on the public dataset while achieving 0.990 on the private dataset. This got me a 73rd placing out of 122 participants, and it met the baseline score of 0.989 as well.
